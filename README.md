# LinguaGemma

An LLM-based chatbot with two distinct prompt-driven teaching logics, benchmarked against each other in a controlled study.

A chatbot's "personality" is really just its prompt-level rules, and different rule sets should produce measurably different outcomes — so we built two prompt logics and compared them directly: a growth-mindset agent and a fixed-mindset control, both teaching the same vocabulary.

We assessed impact using word-search recall tests and thematic/semantic analysis of chat logs. The growth-mindset logic demonstrated higher comfort, confidence, and engagement, winning the **2024 CHI Undergraduate Research Competition**.

- **Publication:** [ACM DL — doi.org/10.1145/3613905.3648628](https://dl.acm.org/doi/10.1145/3613905.3648628)
- **Team:** 3 (Advanced Projects in Interactive Media, Sept 2023 – May 2024)
- **Stack:** Node.js, Express, OpenAI API
